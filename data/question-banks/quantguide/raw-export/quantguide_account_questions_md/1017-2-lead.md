### 题干

Alice and Bob are playing basketball. Both start from $0$ points. Each game, Alice has a $30\%$ chance of winning, independent between games. In each game, the winner gains $1$ point and the loser loses $1$ point. The first person to $2$ points wins the set. Find the probability Alice wins the set. 

### Hint

We can view this is a standard Gambler's Ruin problem that is non-symmetric.

### 解答

We can view this is a standard Gambler's Ruin problem that is non-symmetric. In this case, $p = 0.3, q = 1-p = 0.7,$ and $n = 2$. The formula for the probability that Alice wins is commonly known to be $$\dfrac{1 - \left(\frac{q}{p}\right)^{n}}{1 - \left(\frac{q}{p}\right)^{2n}}$$

We know that $\dfrac{q}{p} = \dfrac{7}{3}$, so plugging this all in, we have that our answer is $$\dfrac{1 - \frac{49}{9}}{1 - \frac{2401}{81}} = \dfrac{81 - 441}{81 - 2401} = \dfrac{9}{58}$$ 

Another way to see this without using the machinery above is to note that we just need to see a sequence of two consecutive wins from a player to win, and this occurs with probability $\left(\dfrac{3}{10}\right)^2 = \dfrac{9}{100}$ for Alice and $\left(\dfrac{7}{10}\right)^{2} = \dfrac{49}{100}$ for Bob. As these are disjoint events in each set of two turns, the probability it occurs for Alice first is $\dfrac{9/100}{9/100 + 49/100} = \dfrac{9}{58}$.
