### 题干

Find the principal branch value of $i^i$. The answer is in the form $e^{-c\pi}$ for a constant $c$. Find $c$.

### Hint

Recall that $i = \cos\left(\dfrac{\pi}{2}\right) + i\sin\left(\dfrac{\pi}{2}\right) = e^{\frac{\pi}{2}i}$.

### 解答

Recall that $i = \cos\left(\dfrac{\pi}{2}\right) + i\sin\left(\dfrac{\pi}{2}\right) = e^{\frac{\pi}{2}i}$. This means that $$i^i = \left(e^{\frac{\pi}{2}i}\right)^i = e^{\frac{\pi}{2}i^2} = e^{-\frac{\pi}{2}}$$ This means that $c = \dfrac{1}{2}$.
