### 题干

How many positive integers at most $1000$ can be written as the sum of some amount of $4$s and/or $5$s? For example, $21 = 4 + 4 + 4 + 4 + 5$

### Hint

Since $4$ and $5$ are relatively prime, in theory, it should be all of them, as we know that the Diophantine equation $4x + 5y = d$ has a solution $(x,y) \in \mathbb{Z} \times \mathbb{Z}$ if and only if $\text{gcd}(4,5) = 1 \mid d$.

### 解答

Since $4$ and $5$ are relatively prime, in theory, it should be all of them, as we know that the Diophantine equation $4x + 5y = d$ has a solution $(x,y) \in \mathbb{Z} \times \mathbb{Z}$ if and only if $\text{gcd}(4,5) = 1 \mid d$. This is not required for this problem, as we can intuitively see that $4 \equiv -1 \hspace{3pt} \text{mod} \hspace{3pt} 5$, so some combination of $4$ and $5$ should yield each integer. All multiplies of $5$ can be formed by just writing it as a sum of some amount of $5$s. Then, we can swap out some of those $5$s for $4$s to get the values between. This only works if there are sufficiently many $5$s to swap out. This is a possibility when there are $< 5$ $5$s in our sum, so we know that every integer $\geq 16$ is possible to be written in this form. We can now manually determine the integers that can't be written in this form are $1,2,3,6,7,$ and $11$. Therefore, $6$ of $1000$ integers can't be written in this form, so $994$ can.
