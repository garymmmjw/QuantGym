### 题干

We run an OLS regression and observe that SSE $= 120$ and SSR $= 80$. What is $R^2$ for this regression?

### Hint

Note that SSR + SSE = SST, so SST = 200.

### 解答

Note that SSR + SSE = SST, so SST = 200. Therefore, $R^2 = 1 - \dfrac{SSE}{SST} = 1 - \dfrac{120}{200} = \dfrac{2}{5}$.
