### 题干

How many non-negative $6-$digit integers have digits that sum to $9$? $116001$ and $801000$ are two examples.

### Hint

The number is in the form $x_1x_2x_3x_4x_5x_6$, where each $x_i$ is a digit. What restrictions, if any, do we have on the $x_i$? Can you relate the $x_i$ to $9$?

### 解答

Our number is in the form $x_1x_2x_3x_4x_5x_6$, where each $x_i$ is a digit. The only restriction we have is that $x_1 \geq 1$ so it is a valid $6-$digit integer. To get a sum of $9$, we are essentially looking for the number of integer solutions to $x_1 + x_2 + \dots + x_6 = 9$ with $x_2,\dots,x_6 \geq 0$, $x_1 \geq 1$. Letting $x_1' = x_1 - 1$, we can see $x_1 = x_1' + 1$ and the restriction is that $x_1' \geq 0$, so we can say we want the number of non-negative integer solutions to $x_1' + x_2 + \dots + x_6 = 8$. This is a classic stars-and-bars question that solves to $\displaystyle \binom{8+6-1}{6-1} = \binom{13}{5} = 1287$
