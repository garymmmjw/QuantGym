# QuantGym Artifacts

This folder stores local QA outputs and one-off extraction artifacts. These files are not runtime dependencies for the web app.

## Folders

- `beta-final-check/`: screenshots from the latest pre-beta desktop and mobile verification pass.
- `regression-screenshots/`: older UI regression screenshots kept for comparison.
- `quantguide-ocr/`: intermediate OCR pages from the QuantGuide import workflow.

Standalone `*-qa.png` files are focused module checks from earlier feature work.
